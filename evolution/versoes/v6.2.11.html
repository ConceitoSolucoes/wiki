S = 26049, Caso o usuário estivesse no formulário de Cadastro de parceiro com parceiro filtrado e um endereço opcional selecionado, ao fazer uma nova busca de dados, as informações não eram exibidas no formulário, e ao tentar fechar a tela de cadastro ocorria o erro "System Error. Code: 1400". Implementado correção na rotina para evitar tal falha. 

Na rotina Cadastro de parceiro, se o usuário estivesse com um parceiro filtrado, caso selecionado um endereço opcional, ao fazer uma nova filtragem de parceiro, os dados da nova busca não eram exibidos no formulário e ao tentar fechar a tela de cadastro ocorria o erro "Code 1400". Implementado  

S = 25953, Implementado validação no Cadastro de equipamento para não permitir ser possível cadastrar um equipamento com um número de série já cadastrado anteriormente. 

 

Melhorias:

S = 26082, A Tela do Cálculo das quantidades apresentada em Pedido/Orçamento passou por melhorias a fim de otimizar a usabilidade e padronizar o sistema. 

Mudanças feitas:

  • Layout redesenhado;
  • Redimensionamento dos componentes baseado no tamanho da tela (Responsivo);
  • Inserido campo "Resultado" a qual exibe o resultado do cálculo prévio sem necessidade de confirmar tela. 

 

S = 24810, A Campanha promocional foi atualizada para contemplar preço de venda por empresa. A partir de agora é possível definir as empresas que vão utilizar a campanha promocional, sendo assim, agora aplica-se o desconto do critério a partir do preço de varejo de cada filial, se assim definido anteriormente no Cadastro de produto, caso não definido, utiliza-se o preço global. Dessa forma cada empresa poderá ter um preço promocional de varejo.

A rotina de Cadastro de Campanha promocional teve seu layout redesenhando para inclusão de novos campos, pois com o uso de filiais, existe a possibilidade de duplicidade de dados, logo esses dados precisam ser revisados, criando assim, uma necessidade de um campo especifico para isso. De toda forma ao iniciar a criação de novo registro a tela se encontra assim : 

(1) Nome da nova campanha;

(2) Código promocional, data inicial e data final da campanha promocional;

(3) As empresas que utilizarão da nova campanha;

(4) As formas de pagamento que englobam a campanha; 

(5) Os critérios da rotina, aonde se adiciona os produtos que faram parte da campanha pelos filtros disponíveis, insere-se a percentual de desconto e as casas decimais do mesmo. Após incluir os itens desejados na lista, deve-se clicar em "Carregar produtos", caso haja duplicidade, o sistema informará : 

 

Caso a opção marcada seja Sim, o sistema retornará a tela exibindo os produtos em duplicidade, logo, cabe ao usuário definir quais dos registros será usado : 

 

Após selecionar quais critérios serão usados nas duplicidades, o sistema exibe a tela com a simulação de como ficará os preços do produto com o desconto aplicado conforme os critérios : 

 

Caso esteja tudo de acordo, é necessário apenas clicar no botão "Gravar promoção" e a campanha promocional será iniciada na data pré-estabelecida.

 

S = 26011, Implementado em Parâmetros gerais, um "combobox: Tipo de Equipamento" com as opções "Veículos", "Equipamento" e "PetShop", afim de gerenciar melhor os dados de cada empresa considerando seu segmento. Selecionado uma opção, modifica-se os descritivos dos campos no cadastro, pois cada segmento necessita de informações diferentes. 

 

 

OBS: Após modificar o tipo de equipamento, é necessário reiniciar o sistema para aplicar a modificação. 

 

S = 26007, Implementado na rotina Painel O.S, a opção de filtragem dos painéis pelos filtros : "Parceiro", "N°. Ordem Serv" e "Data Abertura". 

 

S = 26011, A rotina Painel O.S, possui o recurso de incluir uma nova ordem de serviço (O.S), na tela de inclusão o usuário pode relacionar um equipamento a O.S.. Afim de facilitar o uso da rotina, foi adicionado a opção para cadastrar um novo equipamento a partir do (F8-procurar).

                               Ao clicar no botão novo equipamento, o sistema redireciona a tela de criação do equipamento.

 

Ao criar um novo equipamento por esta opção, o sistema já associa automaticamente o equipamento ao parceiro utilizado, porém é possível editar manualmente essa associação.

Na rotina de Ordem de serviço, dentro de Pedido/Orçamento, também está disponível a opção para cadastrar um novo equipamento a partir do (F8-procurar).

 

Obs: Os para cadastrar um novo equipamento podem mudar de acordo com o tipo de equipamento definido anteriormente nos Parâmetros gerais.
