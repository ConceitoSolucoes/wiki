S = 24242 Disponibilizado uma nova aba para Pagamentos de Títulos no Caixa. Para habilitar essa nova funcionalidade, é necessário ativar a permissão "Pagamentos - Acesso", nas permissões de usuários, conforme imagem abaixo:

 

 


-> A tela de Pagamentos é bem semelhante á tela Baixa/Estorno de Títulos a Pagar, o usuário poderá então fazer pagamentos totais, parciais, globais.

-> Os pagamentos serão feitos usando o saldo do Caixa.

-> Só será possível pagamentos usando Formas de Pagamentos = A vista.

-> Assim como nos Recebimentos, será possível imprimir o recibo após o pagamento.

-> Em caso de Estorno, com Caixa aberto, o valor credita no Caixa, caso o Caixa esteja fechado, deve-se escolher uma conta bancária para que seja creditado o valor.

-> No relatório de Caixa, será detalhado a listagem dos pagamentos, e os valores serão debitados nos totalizadores. 

 

S = 24394, 24395, 24396, 24427 Para melhor controle das operações financeiras do tipo= Título no Força de Venda, foram criadas duas novas permissões de usuários no CIEvolution para faturamento. As validações são necessárias para controle de limite de crédito excedido, e titulos em atraso acima da carência. As permissões devem ser marcadas a critério do cliente, podendo ser para o vendedor que está em rota, já liberando as vendas para impressão no mobile, ou liberar as permissões para o responsável pelo Caixa, para faturamento das vendas no CIE, oriundas do Força de Venda.

Importante: Caso as permissões não sejam liberadas aos vendedores em rota, após confirmar a venda de clientes que possuem as pendencias financeiras informadas acima, será apresentado mensagem informando que a operação não foi concluída por falta de autorizações, não possibilitando ao usuário a impressão da venda. O mesmo ocorrerá com as vendas do Auto Faturamento. Nesses casos, a operação deverá ser concluída pela retaguarda.
